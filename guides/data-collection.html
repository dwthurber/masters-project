Data Collection Methods

The data collection plan defines the researcher's strategy for gathering information from sources, which will become part of the analysis to answer the research question. The goal is to select techniques for data collection that are appropriate to the question. When selecting data collection techniques, keep in mind:

  • What do you want to learn?
  • What has been collected so far?
  • What else do you need to collect and why?
  • Use the worksheet provided in class to help you document.
